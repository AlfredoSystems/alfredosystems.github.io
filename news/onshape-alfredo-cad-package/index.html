MKCad for MiniFRC: The Updated Alfredo CAD Package

Written by Jacob Williams

In FRC, students use an OnShape library of parts called MKCad. New this year, I've made an Onshape Document of MiniFRC parts for a similar purpose, available here!

The recommended way to use this document is to make a copy of it before you even start working on your robot. Then you can design your robot all in a single document. Alternatively, you can insert parts from the Alfredo CAD Package directly into any document.

A final note about the NoU2 and NoU3 CAD, It is recommended that you use the "insert the entire Part Studio as rigid" feature when inserting this part studio into your assemblies.


Happy CADing!
